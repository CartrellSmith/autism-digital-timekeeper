🚀 How to Run the Project
    1: Download or clone the repository
    2: Open index.html in your browser
    3: The clock loads instantly — no build tools required

🧠 JavaScript Highlights
    - Uses a clockApp object to organize all logic
    - setInterval() updates the clock every second
    - formatTime() handles leading zeros + AM/PM logic
    - formatDate() uses toLocaleDateString() for clean formatting
    - updateIcons() switches between day/night themes
    - innerHTML used to animate colon <span> pulse
    - Top‑of‑the‑hour alert triggers when minutes + seconds = 00:00
